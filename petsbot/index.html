Privacy Policy

PetsBot — Last updated: June 2, 2026

1. Information We Collect

PetsBot does not collect, store, or share any personal information. The application uses the Pinterest API solely to publish curated pet product content.

2. How We Use Information

The Pinterest API access token is used exclusively for publishing pins. No user data is collected, analyzed, or shared with third parties.

3. Data Storage

Application credentials are stored locally on the operator's machine and are never transmitted to external servers.

4. Third-Party Services

This application interacts with the Pinterest API. See Pinterest's Privacy Policy for how Pinterest handles data.
