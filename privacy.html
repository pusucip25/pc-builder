Privacy Policy

Last updated: June 2, 2026

1. Information We Collect

PetsBot does not collect, store, or share any personal information from users. The application uses the Pinterest API solely to publish curated pet product content to designated Pinterest boards.

2. How We Use Information

The Pinterest API access token is used exclusively for publishing pins. No user data is collected, analyzed, or shared with third parties.

3. Data Storage

Application credentials (API tokens) are stored locally on the operator's machine and are never transmitted to external servers.

4. Third-Party Services

This application interacts with the Pinterest API. Please refer to Pinterest's Privacy Policy for information about how Pinterest handles data.
